# 🏥 Hospital Resource Utilization Dashboard (Excel)

## 📌 Project Overview

This project is an interactive Excel dashboard designed to analyze hospital patient flow and resource utilization. The dashboard helps visualize key hospital metrics such as patient admissions, department referrals, wait times, gender distribution, age groups, hourly patient arrivals, and monthly trends.

The objective of this project is to demonstrate data cleaning, analysis, and dashboard development skills using Microsoft Excel.

---

## 🎯 Business Objective

Hospital administrators need quick insights into patient flow and operational performance. This dashboard enables them to:

- Monitor total patient visits
- Track admission rates
- Identify peak hospital hours
- Analyze department referrals
- Monitor average patient wait times
- Understand patient demographics
- Observe monthly patient trends

---

## 📊 Dashboard Features

### KPI Cards

- Total Patients
- Total Admissions
- Admission Rate (%)
- Average Wait Time
- Peak (Busy) Hour

### Interactive Filters

- Department
- Gender

### Visualizations

- Department Referrals
- Gender Distribution
- Age Group Distribution
- Hourly Patient Arrivals
- Average Wait Time by Department
- Monthly Patient Trends

### Insights Panel

The dashboard automatically highlights important findings such as:

- Peak hospital hour
- Department with highest referrals
- Highest average wait time
- Overall admission rate
